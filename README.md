# TreeSet API client
 
bit4mation

#### Building

```
yarn
yarn build
```

#### Testing
`yarn test`

#### Testing integration
`yarn test --api http://api-url/api`

#### Generating Test Coverage
`yarn cover`

#### Running

First run backend project
```
cd ../treeset
./gradlew bootRun
```
  
On another console run forntend project (this)  
`yarn start`
  
The browser window should open automatically
